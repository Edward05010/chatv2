# Valentine Bouquet Web App 💐

A cute Valentine's Day web app where people can select flowers, choose a pot, write a message, and send it via email!

## 📁 Project Structure

```
valentine-app/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # All styles and animations
├── js/
│   └── app.js          # All JavaScript functionality
└── images/             # Put your hand-drawn images here!
    ├── rose.png
    ├── tulip.png
    ├── cherry.png
    ├── hibiscus.png
    ├── sunflower.png
    ├── daisy.png
    ├── vase.png
    ├── teapot.png
    ├── honeypot.png
    └── basket.png
```

## 🎨 Adding Your Hand-Drawn Images

1. Draw your flowers and pots
2. Remove the background (make transparent)
3. Save as PNG files with these exact names:
   - **Flowers:** `rose.png`, `tulip.png`, `cherry.png`, `hibiscus.png`, `sunflower.png`, `daisy.png`
   - **Pots:** `vase.png`, `teapot.png`, `honeypot.png`, `basket.png`
4. Put all PNG files in the `images/` folder

That's it! The app will automatically use your drawings!

## ✨ Features

- 🌸 Choose from 6 different flowers
- 🏺 Pick from 4 different pots
- ✍️ Write personalized messages
- 💌 Send to email
- 🎬 Cute drop animation when selecting flowers
- ✨ Sparkle effects
- 👀 Live preview of your bouquet

## 🚀 How to Run

1. Open `index.html` in your browser
2. That's it! No server needed.

## 🎨 Customization

- **Change colors:** Edit `css/style.css`
- **Add more flowers:** Add images to `images/` folder and update `index.html`
- **Modify animations:** Edit the keyframes in `css/style.css`
- **Change functionality:** Edit `js/app.js`

Enjoy! 💕
